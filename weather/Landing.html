Weather vs. Latitude

The Premise of the OpenWeatherMap API Challenge.

To answer a fundamental question:

"Is there a trend in weather characteristics as we approach the equator?"

Temperature, humidity, cloudiness, and windspeed data for a random list of 500 cities from the OpenWeatherMap API was plotted against their latitudes. Each type of data was plotted against the cities latitude. This website provides each visualization, explanations of the data, a comparison page, and the actual source data obtained from the API.
